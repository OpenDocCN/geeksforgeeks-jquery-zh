# jQWidgets jqxTreeGrid 列组属性

> 原文:[https://www . geesforgeks . org/jqwidgets-jqxtreegrid-column groups-property/](https://www.geeksforgeeks.org/jqwidgets-jqxtreegrid-columngroups-property/)

**jQWidgets** 是一个 JavaScript 框架，用于为 PC 和移动设备制作基于 web 的应用程序。它是一个非常强大、优化、独立于平台并且得到广泛支持的框架。****jqxTreeGrid**用于以树状结构表示数据。这个小部件对于显示分层数据的多列、数据分页、排序和筛选、数据编辑、列大小调整、固定列、条件格式、聚合和行选择非常有用。这些小部件还读取和显示来自任何类型数据源的数据，如 XML、JSON、Array、CSV 或 TSV。**

*****列组*** 属性用于为指定的 jqxTreeGrid 设置列组。列组是一组多列标题。**

**可以在“列组”属性中设置属性列表:**

*   ****父组:**这是决定父组名称的字符串属性。**
*   ****名称:**这是决定组名的字符串属性。**
*   ****align:** 这是一个字符串属性，用于确定列标题的对齐方式。可能的值是“左”、“中”或“右”。**

****语法:****

```html
columnGroups: [{
    name: "Group",
    align: "center",
    text: "Organisation"
}]
```

****链接文件:**从给定链接下载 [jQWidgets](https://www.jqwidgets.com/download/) 。在 HTML 文件中，找到下载文件夹中的脚本文件。**

> <link rel="”stylesheet”" href="”jqwidgets/styles/jqx.base.css”" type="”text/css”"> **<脚本类型=“text/JavaScript”src =“scripts/jquery . js”></脚本>
> <脚本类型=“text/JavaScript”src =“jqwidgets/jqxcore . js”></脚本>
> <脚本类型=“text/JavaScript”src =“jqwidgets/jqxdata . js”>>**
