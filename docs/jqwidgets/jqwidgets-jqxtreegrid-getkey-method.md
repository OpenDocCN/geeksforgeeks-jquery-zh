# jQWidgets jqxTreeGrid getKey()方法

> 原文:[https://www . geesforgeks . org/jqwidgets-jqxtreegrid-getkey-method/](https://www.geeksforgeeks.org/jqwidgets-jqxtreegrid-getkey-method/)

**jQWidgets** 是一个 JavaScript 框架，用于为 PC 和移动设备制作基于 web 的应用程序。它是一个非常强大、优化、独立于平台并且得到广泛支持的框架。 **jqxTreeGrid** 用于以树状结构表示数据。这个小部件对于显示分层数据的多列、数据分页、排序和筛选、数据编辑、列大小调整、固定列、条件格式、聚合和行选择非常有用。这些小部件还读取和显示来自任何类型数据源的数据，如 XML、JSON、Array、CSV 或 TSV。

**getKey()** 方法用于返回指定 jqxTreeGrid 小部件的一行的键。如果找不到指定行的键，则返回 null。

**语法:**

```
$("#table").jqxTreeGrid('getKey', row);
```

**参数:**该方法接受如下所示的参数:

*   **行:**这是将要返回其关键字的指定行。

**返回值:**此方法返回指定行的键。

**链接文件:**从给定链接下载 [jQWidgets](https://www.jqwidgets.com/download/) 。在 HTML 文件中，找到下载文件夹中的脚本文件。

> <link rel="”stylesheet”" href="”jqwidgets/styles/jqx.base.css”" type="”text/css”">
> <脚本类型=“text/JavaScript”src =“scripts/jquery . js”></脚本>
> <脚本类型=“text/JavaScript”src =“jqwidgets/jqxcore . js”></脚本>
> <脚本类型=“text/JavaScript”src =“jqwidgets/jqxdata . js”>>
